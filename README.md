# Dictionary App

A simple and responsive Dictionary Application built using **HTML**, **CSS**, and **JavaScript**. The application allows users to search for English words and view their meanings, phonetics, examples, parts of speech, and pronunciation audio.

## Features

* Search for any English word
* Display word definitions
* Show phonetic pronunciation
* Play pronunciation audio
* Display example sentences
* Show parts of speech
* Responsive and user-friendly interface
* Error handling for invalid words

## Technologies Used

* HTML5
* CSS3
* JavaScript 
* Dictionary API

## Installation

1. Clone the repository:

```bash
git clone https://github.com/RainbowPearls/Dictionary-App.git
```

2. Navigate to the project directory:

```bash
cd Dictionary-App
```

3. Open `index.html` in your browser.

## Usage

1. Enter an English word in the search box.
2. Click the **Search** button or press **Enter**.
3. View the word's:

   * Definition
   * Phonetic pronunciation
   * Example sentence
   * Parts of speech
   * Audio pronunciation
